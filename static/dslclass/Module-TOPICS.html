Module 1.1 - Topics

Overview

I currently do not have a detailed breakdown of how the topics will appear week-by-week. The following is a list of topics I would like to cover during this course, but time may not allow covering all of them.

If you have other topics you would like to cover, or particular interests in any specific listed topics, please let me know so I can do my best to cover them.

  • What are DSLs

    • Imperative vs declarative languages
      • Imperative - steps to do something
      • Declarative - intent of what to do

  • Types of DSLs

    • Internal
      • Builders
      • Fluent API
      • Specification
      • Embedded language (ie: SQL)
    • External
      • Generic formats
        • XML
        • JSON
      • Custom formats
        • Delimited
        • Syntax-Based
        • Graphical

  • Uses for DSLs

    • Semantic models
      • By end users
      • By SMEs
      • By programmers
        • Rule systems
        • State machines
        • Macros
        • Interpreters
      • embedded
        • Internal
        • External
      • Standalone
        • Code generation

  • Internal DSLs

    • Expression builder
    • Function sequence
    • Nested function
    • Method chaining
    • Object scoping
    • Annotation
      • Compile-time
      • Runtime

  • External DSLs

    • XML and JSON as DSLs
    • Delimiter-directed translation concepts
    • Syntax-directed translation concepts
      • Vocabulary
      • Grammar
      • Scanner/lexer
      • Parser
      • Parse tree
      • Abstract syntax tree
      • Symbol table
      • Translation
        • Embedded
        • Tree-based
        • Benefits/drawbacks of same-level translation
      • Compilation
        • Benefits/drawbacks of lower-level compilation
      • Interpretation
        • Embedded
        • Tree-based
        • Benefits/drawbacks of interpretation vs translation/compilation
    • Graphical DSL concepts
      • Alternative (non-text) representations
        • Tables
        • Graphs
        • Blocks
      • Combining with Textual Representations
        • Import/export
        • Hand or Machine Editable
      • Examples
        • Blockly (Google/MIT)
        • UML Tools
        • Database E-R Diagrams
      • Benefits/drawbacks of graphical approaches

  • Tools

    • RegEx table
    • Parser Generators
      • LALR
        • lex/yacc
      • LL
        • ANTLR
        • xText
    • Parser Combinator

  • Code Generation

    • Model-Aware
    • Model-Ignorant
    • Tools
      • ANTLR
        • Inline
        • Tree Based
      • xText
        • Text-Based
        • Java Inferencing
      • MPS
    • Templates
    • Generation Gap Pattern
